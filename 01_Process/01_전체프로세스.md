# 전체 프로세스

'음악 만들어줘'로 프로세스 시작

---

## 0단계: 장르-소재 매칭 점검 (전 단계 공통)
- **`00_guide/02_장르소재매칭분석.md` 참조** — 장르의 DNA와 가사 소재가 같은 세계인지 확인
- 장르 출신 일치, 용어 이중성, 감정 동기화, 후크-구조 일치 체크리스트 적용
- 음악유형과 컨셉 선택 시 매칭 적합성을 먼저 판단

## 1단계: 곡 유형 확인/선택 (구현 완료)
- `00_guide/음악유형.md` 확인하여 곡 유형 확정
- `02_MusicType/`에 결과 저장

## 2단계: 곡 컨셉 작성 (구현 완료)
- `00_guide/음악컨셉.md` 확인하여 곡 컨셉 작성
- `03_MusicConcept/`에 결과 저장

## 3단계: Suno 프롬프트 생성 (구현 완료)
- 음악유형 + 음악컨셉을 조합하여 Title / Style / Lyrics 생성
- 제목은 "한글 영문" 형식 (예: "자정의 컴파일 Midnight Compile")
- Style은 200자 이내로 작성
- **Style 작성 시 `00_guide/01_음악프롬프트사운드단어100개.md`의 사운드 단어를 참조** (악기, 리듬, 질감, 보컬, 분위기)
- **가사 작성 시 `00_guide/일본카피도감_분석.md`의 카피 기법 적용** (모순 결합, 결론 생략, 질문형 등)
- **가사 수정 시 `00_guide/사용자의취향_가사가이드.md`의 6가지 패턴 적용** (설명 압축, 명사구 전환, 병렬, 사물의 운명, 소유격 병치, 단어 교체)
- `04_Suno_Prompt/`에 md 파일로 저장
- 참조: `01_Process/03_Suno_ai_프롬프트.md`, `01_Process/02_Suno_입력필드_가이드.md`

## 3.5단계: 가사 자가 비판 및 개선 (구현 완료)
- `00_guide/가사자가비판.md`의 체크리스트(20개 항목)로 가사 점검
- `00_guide/사용자의취향_가사가이드.md`의 6가지 패턴으로 가사 다듬기
- `00_guide/00_통합제작가이드` 기준 적용 (Tell/클리셰/라임/구체성/고전기법 등)
- 위반 사항 발견 시 가사를 직접 수정 (개선된 가사만 프롬프트에 반영)
- **비판 내용은 프롬프트 파일에 작성하지 않음** (Suno가 가사로 인식 방지)
- 모든 위반 해결 후 4단계로 진행

## 3.6단계: 이미지 프롬프트 생성 (구현 완료)
- 곡의 가사·분위기를 반영한 이미지 프롬프트 작성
- 참조: `00_guide/이미지프롬프트작성요령.md`
- `05_image_prompt/`에 md 파일로 저장

## 3.7단계: 유튜브 스크립트 생성 (구현 완료)
- 곡 정보를 바탕으로 유튜브 제목, 설명, 태그 작성
- `08_youtube_script/`에 md 파일로 저장
- **필수 헤딩**: `## 유튜브 제목`, `## 유튜브 설명`, `## 태그` (다른 이름 사용 시 파싱 실패)
- 참조 형식: `08_youtube_script/13_Expiration_Date.md`

> **⚠ 주의**: 3.6, 3.7단계는 반드시 자동화(4단계) 실행 **전에** 완료해야 한다.
> 이미지 프롬프트(`05_image_prompt/`)나 유튜브 스크립트(`08_youtube_script/`)가 없으면
> 이미지 생성이 누락되거나 유튜브 제목이 파일명 기반 fallback으로 잘못 올라간다.

## 4단계: Suno AI 입력 및 생성 (자동화 완료)
- `91_make_mp3/suno_create.py` 실행
- Playwright headless로 Suno Create 페이지에 자동 입력 → 곡 생성
- 참조: `00_guide/Suno_자동화_가이드.md`

## 5단계: mp3 다운로드 (자동화 완료)
- 4단계에서 자동으로 CDN에서 mp3 다운로드
- `05_Mp3/`에 `번호_곡제목_v1.mp3` 형식으로 저장
- Suno는 1회에 2곡 생성 → `_v1`, `_v2`

## 6단계: 이미지 생성 (자동화 완료)
- `92_make_image/image_create.py` 실행
- Gemini API (gemini-2.5-flash-image)로 앨범 자켓 이미지 생성
- v1, v2 각각 별도 이미지 생성
- `06_img/`에 `번호_곡제목_v1.png`, `번호_곡제목_v2.png` (1280x720) 저장

## 7단계: mp4 영상 생성 (자동화 완료)
- `93_make_video/video_create.py` 실행
- v1, v2 각각 개별 mp4 생성 (mp3 1개 + 이미지 1개 = mp4 1개)
- `07_Video/`에 `번호_곡제목_v1.mp4`, `번호_곡제목_v2.mp4` (1280x720) 저장

## 8단계: 유튜브 업로드 (자동화 완료)
- `94_youtube_uploader/youtube_upload.py` 실행
- `07_Video/` mp4 + `08_youtube_script/` 제목/설명/태그 → 미공개 업로드
- v1, v2 각각 개별 업로드, 제목에 V1/V2 자동 추가
- 최초 1회 Google OAuth 인증 필요 (이후 token.json으로 자동 인증)

## 폴더 구조

```
music_gen/
  00_guide/             ← 가이드/설명 문서
  01_Process/           ← 프로세스 문서
  02_MusicType/         ← 음악유형 결과물
  03_MusicConcept/      ← 음악컨셉 결과물
  04_Suno_Prompt/       ← Suno 입력 프롬프트
  05_Mp3/               ← 생성된 mp3
  05_image_prompt/      ← 앨범 자켓 이미지 프롬프트
  06_img/               ← 생성된 이미지
  07_Video/             ← 생성된 mp4
  08_youtube_script/    ← 유튜브 제목/설명/태그
  09_complete/          ← 업로드 완료 기록 (중복 방지)
  91_make_mp3/          ← Suno 자동화 스크립트
  92_make_image/        ← 이미지 생성 자동화 스크립트
  93_make_video/        ← mp4 생성 자동화 스크립트
  94_youtube_uploader/  ← 유튜브 업로드 자동화 스크립트
  99_test/              ← 테스트 스크립트
```

## 넘버링 규칙

같은 곡은 모든 폴더에서 같은 번호를 사용한다:
```
01 = Midnight Compile
  02_MusicType/01_현대노동요.md
  03_MusicConcept/01_새벽코딩.md
  04_Suno_Prompt/01_Midnight_Compile.md
  05_Mp3/01_Midnight_Compile_v1.mp3, _v2.mp3
  05_image_prompt/01_Midnight_Compile.md
  06_img/01_Midnight_Compile_v1.png, _v2.png
  07_Video/01_Midnight_Compile_v1.mp4, _v2.mp4
  08_youtube_script/01_Midnight_Compile.md
  09_complete/01_Midnight_Compile_v1.md, _v2.md
```

## 중복 방지 (존재 시 건너뜀)

| 스크립트 | 확인 대상 | 조건 |
|---------|----------|------|
| `suno_create.py` | `05_Mp3/{곡명}_v*.mp3` | mp3 존재 시 건너뜀 |
| `image_create.py` | `06_img/{곡명}_v1.png`, `_v2.png` | 이미지 존재 시 건너뜀 |
| `video_create.py` | `07_Video/{곡명}_v1.mp4`, `_v2.mp4` | mp4 존재 시 건너뜀 |
| `youtube_upload.py` | `09_complete/{곡명}_v1.md`, `_v2.md` | 완료 파일 존재 시 건너뜀 |

재생성이 필요하면 해당 결과 파일을 삭제 후 스크립트를 다시 실행한다.

## 자동화 실행 방법

```
cd 91_make_mp3 && python suno_create.py      # 곡 생성 + mp3 다운로드
cd 92_make_image && python image_create.py    # 앨범 자켓 이미지 생성
cd 93_make_video && python video_create.py    # mp4 영상 생성
cd 94_youtube_uploader && python youtube_upload.py  # 유튜브 업로드
```
