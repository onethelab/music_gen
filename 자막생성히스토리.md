파이프라인: mp3 → Demucs htdemucs_6s → vocals.wav (캐시) → WhisperX → 갭 그룹핑 → SRT

## 1. 보컬 분리
* Demucs htdemucs_6s 보컬파일 분리

## 2. WhisperX 플랫리스트
* WhisperX transcribe + align → 단어(EN/KO) 또는 문자(JA) 단위 플랫리스트
* VAD onset 값 0.3 (0.5 이상 올리면 정상 가사도 오인식, 환각은 0.9까지 올려도 안 사라짐)
* 환각: JA "ご視聴ありがとうございました", EN "Thank you." — VAD로 해결 불가, 가사 대조에서 필터

## 3. 갭 그룹핑
* 플랫리스트 단어 간 갭으로 구절 단위 그룹핑 (세그먼트와 유사한 단위)
* 갭 기준값 (언어별):
    * 일본어JA: 0.2초 → 15개 구절
    * 영어EN: 0.8초 → 14개 구절
    * 한국어KO: 0.8초 → 20개 구절
* 갭 그룹은 raw 자막 표시용, 가사 매칭의 단위로는 사용하지 않음

## 4. 가사 매칭 (rapidfuzz + concat 방식)
* 플랫리스트 전체를 하나의 문자열(concat)로 합침 + 문자→플랫리스트 인덱스 매핑
* EN/KO: concat에 단어 사이 공백 유지 (단어 경계 오매칭 방지)
* JA: 공백 없이 연결 (원래 붙여 씀)
* rapidfuzz partial_ratio_alignment로 concat에서 가사줄의 최적 위치 탐색
* 매칭된 위치의 플랫리스트 start/end를 가사줄의 타이밍으로 사용 (그룹에 소속시키지 않음)
* 시작 보정: 매칭 시작이 이전 단어 끝부분에 걸리면 다음 단어로 이동
    * 예: "diatom**s** Stain" → "**s**Stain"이 "Stained"로 매칭되어 diatoms 시간(78초)이 start가 되는 문제
    * 보정: 매칭 시작 위치의 첫 글자가 쿼리 첫 글자와 다르면 다음 단어 시작까지 이동

### 매칭 전략: score 우선 점유 + 블랭크
* 전체 가사의 초기 score를 측정하여 score 높은 순으로 점유
* 점유된 영역은 블랭크(마스킹) → 다음 가사가 재사용 불가
* score < 30 매칭은 무의미 → 추정(빨강)으로 전환

### 반복 가사 경합 해결: 인접 인덱스 배정 (proximity 우선)
* 동일 텍스트 가사(예: 1st Chorus #7 = 2nd Chorus #17)는 경합 발생
* 해결: 유니크 가사를 먼저 점유 → 경합 가사는 마스킹 반복으로 모든 후보 위치 탐색
* 각 후보 위치 주변에 이미 매칭된 가사의 인덱스(neighbor)를 확인
* 가사 인덱스와 neighbor의 거리(proximity)가 가장 작은 쌍부터 배정
* 예: 113초 neighbor=#18 → #19(proximity=1)가 #9(proximity=9)보다 먼저 점유
* 결과: 1st/2nd Chorus 자연스럽게 분리

## 5. 후처리
* 겹침 제거: 뒷 가사 start 전에 앞 가사 end
* 추정 가사 배치: 앞뒤 매칭 가사 사이 빈 시간을 **글자 수 비율**로 분배
    * 균등 분배 → 싱크 밀림 발생
    * 글자 수 비율 분배 → 긴 가사는 더 긴 시간, 짧은 가사는 더 짧은 시간 점유
    * 뒤쪽 매칭 없으면 평균 음절당 시간 사용 (fallback)
* score < 30 매칭은 무의미 → 추정(빨강)으로 전환
* 순서 이탈자 검출 (LIS): 시간순 정렬된 가사 인덱스에서 최장 증가 부분수열(LIS) 계산
    * LIS에 포함되지 않는 인덱스 = 이탈자 → 추정으로 전환
    * 예: 시간순 인덱스 [1,3,4,...,16,**2**,17,...] → #2만 역행 → 이탈자 → 추정 배치
    * 나머지가 전부 순서대로인데 하나만 튀는 경우를 자동 감지
* 최소 duration 보장: 2.0초 미만 자막은 다음 자막과 안 겹치는 범위에서 연장
    * 다음 자막이 너무 가까우면 연장 불가 → 원래 duration 유지
* 색상 구분:
    * 파랑: score >= 70 (확실한 매칭)
    * 노랑: score 30~69 (불확실하지만 위치 있음)
    * 빨강: 추정 (WhisperX 미검출, 앞뒤 기준 보간)
