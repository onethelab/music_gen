# 시리즈 제작 가이드

## 이 문서의 목적
시리즈 문서를 작성할 때 장르 간 설명이 복붙처럼 되는 문제를 방지한다.
핵심: **"이 장르에서만 가능한 것"을 쓰고, "아무 장르에나 붙일 수 있는 말"은 쓰지 않는다.**

---

## 1단계: 금지어 체크 — 범용 표현 배제

아래 표현은 여러 장르에 동일하게 적용 가능하므로 시리즈 감성 설명에 사용 금지.
이 표현이 들어가면 장르 구분이 안 된다는 신호다.

| 금지 표현 | 왜 금지인가 |
|-----------|------------|
| 감정은 깊은데 표현은 절제됨 | 고딕신스, 드림팝, 앰비언트팝, 시티팝 전부에 해당 |
| ethereal, breathy (감성 설명으로 쓸 때) | 보컬 스타일에 한정하여 사용은 가능하나 장르 감성 설명에는 금지 |
| deep reverb, wide stereo (감성 설명으로 쓸 때) | 프로덕션 기법이지 장르 특성이 아님. 사운드 키워드에만 사용 |
| 몽환적인, 어둡지만 아름다운 | 너무 넓은 형용사. 구체적으로 뭐가 몽환적인지 써야 함 |
| 반복되는 루프/아르페지오가 최면처럼 | 전자음악 전반에 해당 |

**대신 이렇게 쓴다:**
- "감정 절제" 대신 → 어떤 방식으로 절제하는가? (시티팝: 쿨한 척, 고딕신스: 의식적 숭고함, 드림팝: 안개 속에 녹아듦)
- "어둡지만 아름다운" 대신 → 어떤 종류의 어둠인가? (고딕신스: 성당의 어둠, 다크캬바레: 무대 뒤 어둠, 트립합: 도시 새벽의 어둠)

---

## 2단계: 장르 DNA 추출 — "이것 없으면 이 장르가 아닌 것"

시리즈 감성을 쓰기 전에 아래 질문에 답한다:

### 필수 질문 3가지
1. **이 장르에서 빼면 다른 장르가 되는 요소는?**
   - 시티팝: 슬랩 베이스 + 색소폰 빼면 → 신스인디팝
   - 고딕신스: 합창 패드 + sub bass 빼면 → 다크웨이브
   - 드림팝: 리버스 기타 + 쇼게이즈 벽면 빼면 → 인디팝

2. **이 장르가 표현하는 고독/감정의 종류는?**
   - 시티팝: 풍요 속 공허 (돈은 있는데 외로움)
   - 고딕신스: 선택한 고독 (신이 떠난 성당에 혼자 남은 사람)
   - 드림팝: 부유하는 고독 (잡으려 해도 손에 안 잡힘)
   - 요네즈: 자의식 과잉의 고독 (나를 너무 잘 알아서 외로움)

3. **이 장르의 공간은 어디인가?**
   - 시티팝: 도쿄 시부야 네온거리, 바다가 보이는 해안도로
   - 고딕신스: 버려진 성당, 네온이 비치는 골목
   - 드림팝: 안개 낀 새벽 방 안, 경계가 없는 공간
   - 요네즈: 혼잣말하는 자기 방, 비 오는 횡단보도

---

## 3단계: 비교 대조표 작성

새 시리즈를 만들 때 기존 시리즈와 반드시 비교한다.
감성 설명이 기존 시리즈와 2개 이상 겹치면 다시 쓴다.

### 비교 템플릿

```
| 항목 | [새 장르] | [가장 유사한 기존 장르] |
|------|----------|----------------------|
| 핵심 악기/사운드 | | |
| 고독의 종류 | | |
| 공간 | | |
| 보컬 방식 | | |
| 감정 표현 방식 | | |
| 이 장르만의 한 문장 | | |
```

"이 장르만의 한 문장"이 기존 장르와 바꿔 써도 말이 되면 실패.

---

## 4단계: 시리즈 문서 작성 형식

```markdown
# [장르명] 시리즈

## 장르 DNA (이것 없으면 이 장르가 아님)
* [악기/사운드 레벨에서 이 장르를 정의하는 요소 3~5개]
* [빼면 어떤 장르가 되는지 명시]

## 이 장르만의 감성 (범용 표현 금지)
* [이 장르에서만 성립하는 감성 설명 3~5개]
* [비유를 쓸 경우 이 장르의 공간/세계관에서 가져올 것]

## 유사 장르와의 차이
* vs [장르A]: [구체적 차이점]
* vs [장르B]: [구체적 차이점]

## 사운드 키워드
* [Suno Style에 들어갈 구체적 사운드 단어]

## 보컬
* [보컬 언어, 성별, 구체적 창법]

## 템포/키
* [BPM, 박자, 스케일]

## 시리즈 컨셉 방향 (곡별 소재)
* 01: ...
* 02: ...

## 참고 아티스트
* [레퍼런스]

## 제작 프로세스
* [MusicType 참조 경로, 프로세스 순서]
```

---

## 5단계: 자가 검증 체크리스트

시리즈 문서 작성 후 아래를 점검한다:

- [ ] 금지어 체크: 1단계의 범용 표현이 감성 설명에 없는가?
- [ ] DNA 체크: "장르 DNA" 항목을 빼면 실제로 다른 장르가 되는가?
- [ ] 교체 테스트: 감성 설명을 다른 시리즈에 복붙해도 말이 되면 실패
- [ ] 비교표 작성: 가장 유사한 장르와 대조표를 만들었는가?
- [ ] 한 문장 테스트: "이 장르는 ___이다"를 다른 장르 이름으로 바꿔도 성립하면 다시 쓰기
