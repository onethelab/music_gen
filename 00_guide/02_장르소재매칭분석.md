# 장르-소재 매칭 분석

> 좋은 곡이 나오는 핵심은 **장르의 DNA와 가사의 세계관이 같은 뿌리를 공유**하는 것이다.
> 사례 분석: #10 세이브 포인트 (칩튠 + 게임 소재)

---

## 1. 장르-소재 일치 원칙

장르가 태어난 맥락과 가사의 소재가 같은 세계일 때, 가사가 **비유가 아니라 직역**이 된다. 청자가 사운드를 듣는 순간 자동으로 가사의 세계에 진입한다.

| 원칙 | 설명 | 예시 |
|------|------|------|
| **출신 일치** | 장르가 태어난 문화와 가사 소재가 같은 세계 | 칩튠(게임 음악) + 게임 용어 가사 |
| **사운드 연상** | 악기/질감이 가사의 배경을 자동으로 환기 | 8-bit 사운드 → 게임 화면 연상 |
| **용어의 이중성** | 소재 용어가 일상 감정으로도 읽힘 | "이어하기" = Continue? = 살아가기 |

---

## 2. 용어의 이중성 설계

좋은 가사는 **모든 줄이 두 겹으로 읽힌다.** 소재 세계의 의미와 일상 감정이 동시에 성립해야 한다.

### 사례: 세이브 포인트

| 게임 용어 | 게임 속 의미 | 인생 대응 |
|-----------|-------------|-----------|
| 세이브 포인트 | 저장 지점 | 돌아가고 싶은 과거 |
| 에이치피 바닥 | 체력 소진 | 번아웃 |
| 리셋 버튼 | 다시 시작 | 없는 두 번째 기회 |
| 이어하기 | Continue? | 그래도 살아가기 |
| 경험치 | EXP 획득 | 울고 웃은 시간의 가치 |
| 포션 | 회복 아이템 | 작은 위안 |

---

## 3. 호환성 검사 강화

장르-소재가 잘 매칭된 가사는 **다른 곡에 넣으면 성립하지 않는다.** 이것이 가사의 고유성이다.

- "로드할 수 없는 인생이야" → 연애곡에 불가
- "포션 하나 남았잖아" → 발라드에 불가
- 교환 불가능할수록 좋은 가사

---

## 4. 감정-분위기 동기화

사운드가 주는 감정과 가사가 주는 감정이 **같은 방향**을 가리켜야 한다.

| 음악유형 분위기 | 가사에서의 실현 |
|----------------|----------------|
| Nostalgic | "어제로 돌아갈 수 있다면" — 레트로 사운드 + 과거 회귀 |
| Playful | "보스전 앞에서 떨려도 포션 하나 남았잖아" — 게임 용어의 유머 |
| Bittersweet | "로드할 수 없는 인생이지만 매일 새로운 체크포인트야" — 슬프지만 희망 |

**분위기 키워드를 가사의 감정 설계에 직접 대입**하면 사운드-가사 괴리가 사라진다.

---

## 5. 후크와 장르 구조의 일치

후크의 형태가 장르의 음악적 특성과 맞아야 한다.

| 장르 특성 | 후크 설계 |
|-----------|-----------|
| 칩튠 (짧은 루프 반복) | "이어하기 이어하기" (4음절 반복) |
| 네오소울 (여백, 호흡) | 긴 프레이즈 + 여운 |
| 트로트 (끝음절 강조) | 각운 강한 후크 |

---

## 6. 에너지 곡선과 게임 구조

가사의 감정 흐름이 장르의 에너지 구조와 맞아야 한다.

### 사례: 세이브 포인트의 스테이지 구조

```
Verse 1 [Soft] 지침, 체력 바닥      ← 스테이지 시작
Verse 2 [Gentle] 후회               ← 중간 지점
Chorus [Bright] 이어하기            ← 체크포인트
Verse 3 [Playful] 떨리지만 포션     ← 보스전 앞
Bridge [Whispered] 게임오버 아님    ← 위기 순간
Final Chorus [Powerful] 새 체크포인트 ← 스테이지 클리어
```

## 7. 적용 체크리스트

프롬프트 생성 시 아래를 점검한다:

| # | 항목 | 질문 |
|---|------|------|
| 1 | **출신 일치** | 이 장르가 태어난 문화와 가사 소재가 같은 세계인가? |
| 2 | **이중성** | 소재 용어가 일상 감정으로도 자연스럽게 읽히는가? |
| 3 | **교환 불가** | 이 가사를 다른 장르의 곡에 넣어도 성립하는가? (성립하면 고유성 부족) |
| 4 | **감정 동기화** | 음악유형의 분위기 키워드가 가사 감정과 같은 방향인가? |
| 5 | **후크-구조 일치** | 후크의 길이·반복 패턴이 장르의 음악적 특성과 맞는가? |
| 6 | **에너지 곡선** | 가사의 감정 흐름이 곡 구조의 에너지 변화와 동기화되는가? |

## 8. 장르-소재 매칭 예시

| 장르 | 장르 출신/DNA | 잘 맞는 소재 | 이유 |
|------|-------------|-------------|------|
| 칩튠 | 게임 음악 | 게임, 레트로, 픽셀 | 사운드가 곧 세계관 |
| 네오소울 | 심야 클럽/스튜디오 | 새벽, 라디오, 독백 | 따뜻한 질감 + 고독 |
| 다크 캬바레 | 서커스/극장 | 인형극, 마술쇼, 폐극장 | 기묘한 무대 감성 |
| 셀리송 | 뮤직박스/동화 | 태엽인형, 오르골, 잠자리 | 잔잔하고 반복적인 구조 |
| 트로트 | 한국 서민 문화 | 시장, 택시, 이별 | 한국적 정서의 직접 호소 |
| Lo-fi Hip-hop | 공부/카페 | 비 오는 날, 카페, 새벽 작업 | 배경 음악적 여백 |
