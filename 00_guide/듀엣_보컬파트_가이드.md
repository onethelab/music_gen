# Suno AI 듀엣 보컬 파트 분리 가이드

Suno AI에서 남녀 듀엣 곡 생성 시 보컬 파트가 제대로 나뉘지 않는 문제를 개선하기 위한 가이드.

---

## 1. 핵심 원칙: 3가지 제어 레버

| 레버 | 설명 |
|------|------|
| **매 라인 화자 라벨링** | 모든 가사 라인에 `[Male]`, `[Female]`, `[Both]` 태그 부착 |
| **짧은 섹션 유지** | 섹션당 8~12줄 이내, 긴 섹션은 보이스 드리프트 유발 |
| **섹션 단위 수정** | 전체 재생성 대신, 문제 구간만 Suno의 Replace 기능으로 재생성 |

---

## 2. Style 필드 작성법

Style 필드에 듀엣 패턴과 보컬 특성을 명시한다. **1~2줄로 간결하게** 작성.

### 듀엣 패턴 4가지

| 패턴 | 설명 | 적합한 상황 |
|------|------|------------|
| **Duet (Alternating)** | 한 줄씩 번갈아 부르기 | 대화체 가사, 싸움/토론 |
| **Duet (Call & Response)** | 리드 + 3~6단어 응답 | 질문/대답 구조 |
| **Duet (Harmony Hook)** | 솔로 Verse + 함께 Chorus | 일반적인 듀엣 |
| **Lead + Adlibs** | 메인 보컬 + 괄호 안 추임새 | 한 쪽이 메인인 경우 |

### Style 필드 예시

```
K-Pop Duet (Alternating), R&B, 110 BPM, Female lead airy and intimate, Male lead grounded and steady, Clear turn-taking each line, One shared hook in chorus only, no soft ballad
```

### 보컬 대비 강화 키워드

두 보컬이 비슷하게 들리는 것을 방지하려면 **음색 대비 수식어**를 추가:

| Male 수식어 | Female 수식어 |
|------------|--------------|
| grounded, deep, husky | airy, bright, sweet |
| raspy, rough, powerful | smooth, silky, delicate |
| baritone, low-pitched | soprano, high-pitched |
| steady, firm | flowing, light |

---

## 3. Lyrics 필드 작성법 (가장 중요)

### 규칙 1: 모든 라인에 화자 태그 달기

상단에 한 번만 선언하면 안 됨. **매 라인/섹션마다 반복** 표기해야 보이스 교체가 안정적.

**나쁜 예 (보이스 드리프트 발생):**
```
[Verse 1]
[Male]
첫 번째 줄
두 번째 줄
세 번째 줄
네 번째 줄
```

**좋은 예 (안정적 파트 분리):**
```
[Verse 1]
[Male Vocal]
첫 번째 줄
두 번째 줄

[Female Vocal]
세 번째 줄
네 번째 줄
```

### 규칙 2: [Both] 태그는 최소한으로

두 사람이 동시에 전체 문장을 부르면 Suno가 혼란. **Chorus의 짧은 훅 1개**에만 사용.

**나쁜 예:**
```
[Chorus]
[Both]
긴 첫 번째 줄을 함께 부름
긴 두 번째 줄도 함께 부름
긴 세 번째 줄도 함께 부름
긴 네 번째 줄도 함께 부름
```

**좋은 예:**
```
[Chorus]
[Male Vocal]
첫 번째 줄
[Female Vocal]
두 번째 줄
[Both, Harmonized]
짧은 훅 한 줄만 함께
[Male Vocal]
네 번째 줄
```

### 규칙 3: 괄호로 배경 보컬/추임새 표현

```
[Female Vocal]
내가 슬프다잖아 그게 근거야
(그게 근거냐고?)
```
괄호 안의 텍스트는 배경 보컬이나 추임새로 처리됨.

### 규칙 4: 성별 특정 가사 회피 (가장 중요)

Suno는 남녀 보컬 파트를 100% 정확히 분리하지 못한다. 의도와 다른 성별이 부를 수 있으므로, **누가 불러도 어색하지 않은 가사**를 작성해야 한다.

**나쁜 예 (성별 특정적):**
```
나 머리 말리다 봤는데          ← 여자만 할 법한 말
로봇이랑 연애하나 봐           ← 여자가 남자에게 하는 뉘앙스
오빠가 그런 식이면             ← 여자만 쓰는 호칭
군대 갔다 와서 달라졌네         ← 남자만 해당
```

**좋은 예 (성별 무관):**
```
왜 자꾸 엉뚱한 걸 원해         ← 누가 해도 자연스러움
같은 말인데 다른 세상이야       ← 성별 무관
해결책 말고 한마디만 해줘       ← 누구나 할 수 있는 말
난 다 너를 위해 하는 말인데     ← 성별 무관
```

**원칙:**
- 성별 특정 호칭 금지: 오빠, 누나, 형, 언니 등
- 성별 특정 행동/경험 금지: 머리 말리기, 화장, 군대 등
- 상대를 성별로 지칭하는 표현 금지: "남자가 다 그래", "여자는 왜 그래" 등
- 감정, 생각, 상황 중심의 **중성적 가사**로 작성

### 규칙 5: 대문자로 강조/소리침 표현

```
[Male Vocal, Shouted]
아니 내가 뭘 잘못했는데
팩트만 말한 건데 왜 울어
```

## 4. 보컬 태그 레퍼런스

### 화자 지정 태그
```
[Male Vocal]    [Female Vocal]    [Both]
[Male Singer]   [Female Singer]   [Duet]
[Male Voice]    [Female Voice]    [Unison]
[Male Lead]     [Female Lead]     [Harmonized]
```

### 보컬 강도 태그
```
[Whispered]  [Soft]  [Gentle]  [Spoken]
[Powerful]   [Belted]  [Shouted]  [Screamed]
```

### 보컬 스타일 태그
```
[Falsetto]  [Breathy]  [Raspy]  [Smooth]
[Soulful]   [Conversational]  [Aggressive]
```

### 래핑/스피킹 태그
```
[Rapped]  [Fast Rap]  [Melodic Rap]
[Spoken Word]  [Talk-Sing]
```

### 조합 사용 (태그 중첩)
```
[Female Vocal, Whispered]
사실 손 잡고 싶었는데

[Male Vocal, Spoken, Soft]
나도 그랬는데
```

## 5. 듀엣 친화적 장르

아래 장르를 Style에 포함하면 Suno가 듀엣 패턴을 더 잘 인식:

| 장르 | 이유 |
|------|------|
| **Broadway / Musical Theater** | 듀엣이 가장 흔한 장르, 파트 분리 학습 데이터 많음 |
| **K-Drama OST** | 남녀 듀엣 패턴이 많이 학습됨 |
| **R&B Duet** | 보컬 교차 패턴이 자연스러움 |
| **Country Duet** | 전통적으로 남녀 듀엣이 많은 장르 |
| **Opera / Operatic** | 성악 듀엣의 명확한 파트 구분 |

## 6. 자주 발생하는 문제와 해결

### 문제 1: 남녀 목소리가 구분이 안 됨
- **원인**: Style에 보컬 대비 수식어 없음
- **해결**: `Male baritone grounded` / `Female soprano airy` 같은 음색 대비 추가
- **해결**: 장르에 `Musical Theater` 또는 `Broadway` 추가

### 문제 2: 중간에 목소리가 바뀜 (보이스 드리프트)
- **원인**: 섹션이 너무 길거나, 화자 태그가 섹션 시작에만 있음
- **해결**: 섹션을 8줄 이내로 줄이고, 2~4줄마다 화자 태그 반복

### 문제 3: [Both] 구간에서 한 목소리만 나옴
- **원인**: Suno의 한계, 완전한 유니슨 보장 불가
- **해결**: `[Both, Harmonized]`로 명시하거나, `[Duet]` 태그 사용
- **대안**: [Both] 대신 `[Female Vocal]` 줄 + `(남성 화음)` 괄호 조합

### 문제 4: 목소리는 다른데 파트 타이밍이 어긋남
- **원인**: 가사가 너무 길어 리듬이 밀림
- **해결**: 한 줄당 10~15자 이내로 짧게 유지

## 7. 개선된 듀엣 Lyrics 템플릿

```
[Intro]
[Instrumental, Piano stabs, 808 Bass]

[Verse 1]
[Male Vocal, Conversational]
짧은 가사 한 줄
짧은 가사 두 줄

[Verse 2]
[Female Vocal, Sharp]
짧은 가사 한 줄
짧은 가사 두 줄

[Chorus]
[Male Vocal, Powerful]
코러스 첫 줄
[Female Vocal, Powerful]
코러스 둘째 줄
[Both, Harmonized]
함께 부르는 핵심 훅 한 줄
[Male Vocal]
코러스 마지막 줄

[Bridge]
[Female Vocal, Whispered]
브릿지 한 줄
[Male Vocal, Whispered]
브릿지 응답 한 줄

[Final Chorus]
[Both, Belted]
최종 코러스 핵심 훅
```
